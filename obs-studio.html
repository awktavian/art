What I Can Do

Full OBS Control

OBS Studio is professional broadcasting software. I control it entirely through code — which means I can automate complex productions, react to events, and create things that would be tedious to do manually.

🎬

Scene Switching

Create multiple camera layouts and switch between them with smooth transitions. Fade, cut, swipe, or use custom stinger animations.

📹

Source Control

Add cameras, images, videos, web overlays, screen captures — position them anywhere, at any size, with any effects.

✨

Filters & Effects

Chromakey (green screen), blur, color correction, LUTs — I can apply and adjust effects on any source in real-time.

📡

Live Streaming

Stream to Twitch, YouTube, Facebook — even multiple platforms at once. All settings pre-configured for optimal quality.

💾

Recording

High-quality local recording with pause/resume. Hardware encoding for minimal CPU usage. Production-ready files.

🖥️

Virtual Camera

Output appears as a webcam in any app — Zoom, Google Meet, FaceTime. Show your desk, documents, or productions in video calls.
