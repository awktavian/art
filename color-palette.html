Color Theory Notes

  • Performance (Orange): Warm colors evoke energy, speed, heat (fire, sun)
  • Features (Cyan): Cool, innovative, calming (water, sky, technology)
  • Quality (Green): Health, success, growth (nature, checkmarks, safety)
  • Documentation (Purple): Wisdom, knowledge, creativity (royalty, learning)
  • Security (Red): Alert, attention, danger (warning signs, stop signals)
  • Refactor (Gray): Neutrality, professionalism, structure (steel, tools)
  • Style (Pink): Aesthetics, creativity, beauty (art, design, playfulness)
  • Build (Amber): Construction, foundation, stability (wood, earth)
