# Design Skill

Creating visual interfaces.

## When to Use

- Building HTML pages
- Creating dashboards
- Making visualizations

## Key Files

| File | Purpose |
|------|---------|
| `CRAFT_PRINCIPLES.md` | Typography, spacing, color |

## Quick Reference

### Colors

```css
/* Backgrounds */
--void: #0a0a0f;
--obsidian: #12121a;

/* Status */
--pass: #00ff88;
--fail: #ff4444;
--warn: #ffd700;
```

### Typography

```css
--font-display: 'Playfair Display', serif;
--font-body: 'Crimson Pro', serif;
--font-mono: 'Berkeley Mono', monospace;
```

### Spacing (8px grid)

```css
--space-1: 8px;
--space-2: 16px;
--space-3: 24px;
--space-4: 32px;
```
