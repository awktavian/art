# Tool Time University

A Canvas LMS course celebrating the cultural impact of *Home Improvement* (1991-1999).

## Course Overview

Tool Time University is a fun, educational exploration of the beloved ABC sitcom *Home Improvement*, starring Tim Allen. The course examines the show's cultural significance, comedic techniques, character development, and themes of masculinity, family, and personal growth.

## Course Structure

### Grading System: Home Equity Dollars

Instead of traditional points, students earn **Home Equity Dollars (HED)** - inspired by the *Home Improvement* board game. This thematic currency makes grading more engaging while maintaining academic rigor.

| Activity Type | Base Value | Description |
|---------------|------------|-------------|
| Lessons | 50 HED | Reading + comprehension check |
| Assignments | 100 HED | Rubric-graded creative projects |
| Quizzes | 75 HED | Multiple choice, 2 attempts allowed |
| Discussions | 50 HED | Initial post + 2 peer replies |

### Fun Facts Featured in Module 1

- **Original title:** "Hammer Time" (changed due to MC Hammer legal concerns)
- **Original Jill:** Frances Fisher was cast before Patricia Richardson
- **Al's origin:** Richard Karn was hired for one episode, then became permanent
- **Tim Allen's background:** Stand-up comedian whose "Men Are Pigs" routine inspired the show
- **Premiere date:** September 17, 1991
- **Viewership:** 16.9 million viewers for the pilot
