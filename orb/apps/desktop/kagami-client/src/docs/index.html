Core Principles

Light as Material

The design system treats light as a physical material. When light passes through glass, it bends (refraction) and separates into its component colors (dispersion). Our components simulate this behavior with chromatic edge effects and caustic backgrounds.

Fano Spectral Mapping

The color system is mathematically derived from the Fano plane—a structure with 7 points and 7 lines where every pair of lines intersects at exactly one point. Each point maps to a spectral wavelength:

Glassmorphism Foundation

All surfaces use translucent glass with backdrop blur and saturation. This creates depth and context-awareness while maintaining readability.

Motion with Meaning

Animations use standard durations (89ms, 144ms, 233ms, 377ms, 610ms, 987ms) and natural easing curves (ease, ease-out, ease-in, spring).

Installation

Include the three core CSS files in your project:

<link rel="stylesheet" href="css/prism-tokens.css"> <link rel="stylesheet" href="css/prism-components.css"> <link rel="stylesheet" href="css/prism-effects.css">

Then apply component classes to your HTML:

<button class="prism-button prism-button--primary"> Click me </button> <div class="prism-card"> <h3 class="prism-card__title">Title</h3> <p class="prism-card__content">Content</p> </div>

Design Tokens

All values are exposed as CSS custom properties for easy customization:

Category Example Token Description
Colors --prism-e1 to --prism-e7 Spectral colors (red to violet)
Glass --prism-blur, --prism-opacity Backdrop blur and transparency
Timing --prism-dur-fast, --prism-dur-slow Standard durations
Easing --prism-ease-cusp, --prism-ease-spring Standard easing curves
Radius --prism-radius-sm to --prism-radius-xl Border radius scale
