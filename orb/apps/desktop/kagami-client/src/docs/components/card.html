Card

Glass containers with chromatic dispersion edges. Cards are the primary surface for grouping content in the design system.

Live Preview

Hover over the card to see the chromatic dispersion effect expand.

Card Title

Supporting text here

This is the card content area. On hover, the red-shifted and blue-shifted chromatic edges expand outward, creating a prismatic dispersion effect.

Code

<div class="prism-card"> <div class="prism-card__header"> <h3 class="prism-card__title">Card Title</h3> <p class="prism-card__subtitle">Supporting text</p> </div> <div class="prism-card__content"> <p>Card content here.</p> </div> <div class="prism-card__footer"> <button class="prism-button prism-button--primary">Action</button> </div> </div>

Chromatic Dispersion

The signature glass effect. Two pseudo-elements create red-shifted (warm) and blue-shifted (cool) edges that expand on hover.

/* Red-shift (warm edge) - ::before */ .prism-card::before { background: var(--prism-gradient-red-shift); inset: calc(var(--prism-dispersion) * -1); /* 3px */ } /* Blue-shift (cool edge) - ::after */ .prism-card::after { background: var(--prism-gradient-blue-shift); inset: calc(var(--prism-dispersion) * -1); /* 3px */ } /* On hover, dispersion increases */ .prism-card:hover::before, .prism-card:hover::after { inset: calc(var(--prism-dispersion-hover) * -1); /* 6px */ }

Disabling Dispersion

For performance-critical sections, disable the chromatic edges:

<div class="prism-card prism-no-dispersion"> <!-- Plain glass card without chromatic edges --> </div>

Accessibility

Accessibility Notes
  • Semantic structure: Use appropriate heading levels within cards.
  • Interactive cards: If the entire card is clickable, use <article> or <button> with proper focus states.
  • Focus visibility: Cards that receive focus should have visible focus indicators.
  • Color contrast: Text on glass surfaces maintains WCAG AA contrast ratios.
  • Reduced motion: Chromatic animations are disabled with prefers-reduced-motion.

Platform Variants

Desktop

Full chromatic dispersion effects with backdrop blur. Hover animations enabled.

AR / VisionOS

Increased dispersion spread (5px base) and lower glass opacity (0.4) for spatial computing. Cards should feel like floating glass panels.

/* VisionOS overrides */ @media (display-mode: standalone) { :root { --prism-blur: 40px; --prism-opacity: 0.4; --prism-dispersion: 5px; } }

Watch

Simplified cards with reduced blur and minimal dispersion for small screens and battery efficiency.

Do's and Don'ts

Do
  • Use cards to group related content
  • Maintain consistent card sizes in grids
  • Use colony colors for semantic meaning
  • Place actions in the footer area
  • Use proper heading hierarchy
Don't
  • Nest cards within cards
  • Use solid backgrounds on cards
  • Overcrowd cards with too much content
  • Mix multiple colony colors in one card
  • Use cards for single-line content
