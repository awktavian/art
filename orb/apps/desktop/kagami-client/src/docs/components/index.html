Component Anatomy

All The components share common structural elements:

Layer Purpose CSS
Glass Surface Translucent background with blur backdrop-filter: blur(20px)
Red-shift Edge Warm chromatic dispersion (::before) --prism-gradient-red-shift
Blue-shift Edge Cool chromatic dispersion (::after) --prism-gradient-blue-shift
Border Subtle glass edge border: 1px solid var(--prism-border)
Content Text and child elements color: var(--prism-text)
