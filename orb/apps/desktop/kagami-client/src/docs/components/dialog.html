Dialog

Modal overlays with glass surfaces and backdrop blur. Use for focused interactions that require user attention.

Live Preview

Click the button to open the dialog.

Dialog Title

This is the dialog content. The backdrop uses blur and the modal itself has chromatic edges for the signature glass effect.

Press Escape or click outside to close.

Code

<div class="prism-modal-backdrop prism-modal-backdrop--open"> <div class="prism-modal"> <div class="prism-modal__header"> <h3 class="prism-modal__title">Dialog Title</h3> <button class="prism-modal__close">...</button> </div> <div class="prism-modal__content"> <p>Content here.</p> </div> <div class="prism-modal__footer"> <button class="prism-button">Cancel</button> <button class="prism-button prism-button--primary">Confirm</button> </div> </div> </div>

Accessibility

Accessibility Notes
  • Use role="dialog" and aria-modal="true"
  • Set aria-labelledby to the modal title
  • Trap focus within the modal when open
  • Return focus to trigger element on close
  • Support Escape key to close
