Colors

The color system is derived from the Fano plane, mapping seven mathematical points to seven spectral wavelengths.

Spectral Colors

Each color corresponds to a visible light wavelength, from red (620nm) to violet (400nm). Click any color to copy its token name.

Brand Assignment

Use the data-colony attribute to assign a spectral color to a component. This sets the --colony-color variable.

<div data-colony="beacon"> <!-- Uses --prism-beacon as --colony-color --> </div> <button class="prism-button--primary" data-colony="spark"> Red Button </button>

Dark Mode Adaptation

Colors automatically adjust in dark mode for better visibility. Spectral colors become slightly brighter, and surface/text colors invert.

/* Dark mode is automatic via prefers-color-scheme */ @media (prefers-color-scheme: dark) { :root { --prism-e1: #FF5A50; /* Brighter red */ --prism-surface: rgba(0, 0, 0, 0.7); --prism-text: rgba(255, 255, 255, 0.92); } }

Usage Guidelines

Do
  • Use spectral colors for accents and highlights
  • Apply alpha variants for subtle backgrounds
  • Use colony colors for semantic meaning
  • Let surfaces be translucent when possible
Don't
  • Use full spectral colors for large surfaces
  • Mix too many spectral colors in one view
  • Use colors outside the defined tokens
  • Ignore the dark mode adaptations
