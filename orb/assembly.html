Step 1.2: Install Maglev Module

CRITICAL: Do not power on yet

  1. Remove protective film from maglev coils
  2. Apply thermal pad to bottom of control board
  3. Place maglev module centered in enclosure
  4. Mark mounting holes
  5. Secure with M3 screws (do not overtighten)

Test (unpowered):
- Place a magnet on top
- Should feel strong attraction
- Magnet should center naturally

Step 1.3: Install Resonant Transmitter

NOTE: Standard Qi EPP will NOT work at 15mm gap with maglev magnets.
Custom resonant coils (80mm Litz wire @ 140kHz) are required.

  1. Position resonant TX coil (80mm) in center of maglev area
  2. Install ferrite shield (Mn-Zn, 90mm) BETWEEN coil and maglev magnets
  3. Coil should be 2-3mm below maglev surface
  4. Route coil wires to bottom compartment
  5. Secure with hot glue (temporary) or brackets

Phase 3: Mechanical Assembly (3 hours)

Step 3.1: Prepare Shell Components

  1. Clean acrylic hemispheres with IPA
  2. Apply mirror film to inner hemisphere:
    - Cut film slightly oversized
    - Spray adhesive application
    - Smooth with squeegee
    - Trim edges
  3. Verify infinity effect with LED test

Step 3.2: Install Internal Frame

  1. Press-fit threaded inserts into frame (heat with soldering iron)
  2. Mount CM4 bracket to frame (M2.5 screws)
  3. Mount CM4 module to bracket
  4. Route antenna (if external WiFi)

Step 3.3: Install Battery

  1. Line battery pocket with foam
  2. Insert battery pack
  3. Secure with velcro straps
  4. Route battery cable to BMS

Step 3.4: Install LED Ring

  1. Snap LED mount ring onto frame tabs
  2. Insert SK6812 ring into groove
  3. Insert diffuser ring above LEDs
  4. Route LED cable to main PCB

Step 3.5: Install Resonant Receiver

  1. Mount resonant RX coil (80mm Litz) to bottom of frame
  2. Install ferrite shield (Mn-Zn, 90mm) BETWEEN coil and frame
  3. Align with center axis
  4. Route coil cable to BMS

Step 3.6: Apply Thermal Compound

CRITICAL: Proper thermal contact required

  1. Clean CM4 SoC surface with IPA
  2. Apply thin layer of thermal compound (Arctic MX-4 or similar)
  3. Attach heatsink to CM4
  4. Verify good contact (heatsink should not wobble)

Step 3.7: Final Cable Management

  1. Organize all cables within frame
  2. Use cable ties or tape to secure
  3. Ensure no cables will be pinched during shell closure
  4. Verify no cables block thermal path

Step 3.8: Close Shell

  1. Power on electronics one last time to verify
  2. Power off
  3. Lower outer hemisphere over frame
  4. Align with inner hemisphere
  5. Secure with clips or adhesive (temporary for testing)

Phase 4: Integration Testing (2 hours)

Step 4.1: First Float Test

  1. Place orb on base station
  2. Observe levitation:
    - Should rise smoothly (3 seconds)
    - Should stabilize at ~15mm
    - Should not oscillate

If issues:
- Weight too heavy: Reduce battery or use lighter shell
- Unstable: Adjust maglev PID settings
- Won't rise: Check magnet orientation

  1. While floating, check charging indicator
